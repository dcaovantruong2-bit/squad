# CM93 Retro

A reusable Open Design package for CM93 Retro.

## Product Overview

Authentic Championship Manager 93 Amiga/PC aesthetic — extracted from actual game screenshots. Yellow title bars, blue panels, grey content areas, 1px beveled borders, flat solid colors, Amiga Topaz-style pixel font. This design-system package is designed for product, app, workspace, and platform surfaces that need a reusable visual direction rather than a one-off mockup. It provides a concrete token layer, focused preview cards, and an applied UI kit so future agents can build interfaces with the same hierarchy, density, component roles, and interaction rules captured in DESIGN.md.
